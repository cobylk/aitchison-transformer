This hosts a transformer with the residual stream constrained to lie on the standard simplex, for interpretability -- see [this blog](https://www.lesswrong.com/posts/kjL9req2p79nSNe5H/interpretable-by-design-constraint-sets-with-disjoint-limit) for some intuitions as to why this was worth trying. 

This is a toy repo with an isolated component I wrote from a larger project (extension of SPAR) whose codebase is not public; it's here to showcase the implementation for portfolio reasons.
